# SchemaValidation - Exemplos (JSON Schema)

Exemplos mínimos de validação de JSON Schema com foco em:

- validação profunda (subschemas)
- `SchemaPath`
- `$ref` local e por arquivo

## Arquivos

- `SchemaRefAndSchemaPathDemo.dpr`: demo console
- `schemas/`: schemas usados no demo

## Como executar

1) Compile o `SchemaRefAndSchemaPathDemo.dpr`.
2) Execute o binário gerado.
